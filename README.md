# To-Do List API

Aplicación para la gestión de tareas desarrollada con Node.js, Express y MongoDB.
Permite agregar, listar, editar, completar y eliminar tareas.

## Tecnologías utilizadas
- Node.js
- Express.js
- MongoDB
- Mongoose
- Render (despliegue)

## Despliegue
La aplicación se encuentra desplegada y funcionando en la siguiente URL:

https://todolist-2peg.onrender.com/

## Instalación y ejecución local

1. Clonar el repositorio:

Instalar dependencias:

npm install


Crear un archivo .env en la raíz del proyecto con las siguientes variables:

PORT=3000
MONGODB_URI=tu_cadena_de_conexion_mongodb


Ejecutar el proyecto:

npm run dev


Acceder desde el navegador:

http://localhost:3000
